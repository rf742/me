Groff

Groff is a pretty cool document preparation system. I have found much use for it as a lightweight alternative to TeX or a word processor. It is built into most unix-like operating systems (macOS, BSD, linux...) and is also available for windows.

It features very terse commands, and is ideal for not only interactive document writing, but for use in generating automated reports, as it is processed in a single pass by the compiler, and thus can be used in UNIX pipes.
